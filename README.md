To-Do List Application

Simple task management web application built using HTML, CSS and JavaScritp.

Features:
- Add new tasks
- Mark tasks as completed
- Edit tasks by double-clicking
- Remove tasks
- Filter tasks (all / completed / pending)
- Counter displaying pending and completed tasks
- Data persistence using Local Storage

Technologies:
- HTML
- CSS
- JavaScript

Purpose:
This project was developed to practise DOM manipulation, event handling and basic front-end logic.
